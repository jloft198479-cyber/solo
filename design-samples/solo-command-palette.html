产品愿景

solo 是一个本地优先的 Markdown 编辑器，面向中文沉浸式写作。它不依赖云端、不上传你的文字，打开即写，关掉即走。

从大纲导航到命令面板，所有能力都围绕「少动手、多思考」设计。

写作应当是安静的。我们把所有干扰都移到了视线之外，只留下你和文字。

命令面板说明（对应设计评审 ⑥）

复用现有命令体系：solo 已有完整 command registry + dispatcher（findCommandByShortcut / useCommandDispatcher），命令面板只是把它「可视化 + 可搜索」，零架构成本。
键盘流定位：默认 Ctrl/⌘ + K 唤起，模糊匹配 + 分组 + 右侧快捷键提示，让「记不住快捷键」和「找不到菜单」同时解决。
视觉语言统一：浮层、圆角、阴影、激活态全部沿用 solo 既有的 popover token，与浮动格式菜单 / 斜杠菜单同一套。当前高亮项即「保存当前文档」这一条。
