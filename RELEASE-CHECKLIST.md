## 功能回归（修什么测什么 + 关联面）

- [ ] **新建空白文档** — 打开即写，无报错

## 路径/存储

- [ ] **未保存文档** 拖图/贴图 → 提示先保存
- [ ] **已保存文档** 拖图 → 存到 `assets/` → 相对路径写入
- [ ] **已保存文档** 贴截图 → 存到 `assets/` → 相对路径写入
- [ ] **自定义存储路径** 贴图 → 走 `asset://` 协议 → 正常显示
- [ ] **重命名文档** → `assets/` 目录下的图片引用是否同步（不要求自动迁移，检查是否有预期行为）
