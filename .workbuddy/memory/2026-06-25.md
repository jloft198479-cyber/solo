## 焦点模式修复

问题：切换焦点模式后无可见效果。
根因：ProseMirror plugin 依赖 meta 事务触发重算，而 `dispatch` 空事务在部分情况下无法 reliably 触发 decorations 重绘。
方案：
- 简化 `paragraph-focus.ts`：移除 plugin state 缓存，decorations props 实时根据 `html.focus-mode` class 与当前段落计算
- `MarkdownEditor.vue` watch `isFocusMode`，切换后调用 `view.updateState(view.state)` 强制 ProseMirror 重绘 decorations
- 保留 `paragraph-active` / `paragraph-dimmed` CSS，当前段落实心、其余淡出至 22%
结果：
- `vue-tsc` 通过
- 200 tests 全过
- Vite build 通过
